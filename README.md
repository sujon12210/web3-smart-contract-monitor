# Web3 Smart Contract Monitor

This project watches a smart contract.
It prints events to the screen.
You can learn how blockchain events work.

This project is made for beginners.
All code is simple and clear.

## Features
- Listen to contract events
- Print event data
- Ethereum testnet support
- Simple command line tool

## Requirements
- Node.js installed
- RPC provider account
- Contract address and ABI

## Setup
1. Clone this repo
2. Open folder in terminal
3. Run: npm install
4. Create a file called .env
5. Add RPC URL and contract address
6. Run: node monitor.js

## Example
Run the script.
Trigger contract actions.
Events will show in terminal.

## Learning Goals
- Learn blockchain events
- Learn real time listeners
- Learn smart contract logs
